# Sass/SCSS Practice Files

This directory contains various SCSS files designed to demonstrate and practice the features of Sass, a powerful CSS preprocessor.

## File Overview

- **Variables**:
  - `1-color_variable.scss`, `2-color_variables.scss` - Examples of SCSS variables for color management.

- **Nesting**:
  - `3-nested_tag.scss`, `4-nested_class.scss`, `5-nested_child.scss`, `6-nested_hover.scss`, `7-nested_deeper.scss` - SCSS nesting for organizing styles hierarchically.

- **Mixins & Extends**:
  - `8-mixin_margins.scss` - Demonstrates reusable code blocks using mixins.
  - `9-extend_list.scss` - Shows code inheritance using `@extend`.

- **Loops & Iterations**:
  - `11-loop_photos.scss`, `12-loop_header.scss`, `100-loop_col.scss` - Examples of loops for automating repetitive tasks in SCSS.

- **Media Queries**:
  - `101-media_query.scss`, `102-media_query.scss` - Examples of responsive design using SCSS media queries.

## Compilation

To compile any SCSS file into CSS:
```bash
sass <filename>.scss <output>.css
